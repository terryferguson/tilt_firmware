void MotorController::extend ( )
inline

Extends the motorized system.

◆ home()

void MotorController::home ( )
inline

Function to control the homing.

This function retracts, checks leader and follower out of range status, and updates the motor status until both motors are out of range. It then zeros out the positions of the motors and resets the soft movement status

< Set the PID flag to true

< Reset the soft movement

< Set the outOfRange flag to false for all motors

< Set the command to retract for all motors

< Set the system direction to retract

< Set the requested direction to retract
